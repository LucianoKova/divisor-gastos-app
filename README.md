📌 Project Overview

Divisor de Gastos is a web-based application built in Python that solves a real-world financial problem: splitting shared expenses efficiently and transparently among multiple participants.

The system automatically calculates individual balances and optimizes debt distribution, minimizing the number of transactions required to settle accounts.

This project demonstrates backend logic design, financial computation modeling, and deployment to a production environment.

🧠 Problem Statement

When multiple people share expenses (trips, rent, dinners), manual calculation becomes inefficient and error-prone.

This application:

Tracks who paid each expense

Determines how much each participant should contribute

Calculates net balances

Identifies who owes whom

Reduces friction in financial settlements

⚙️ Technical Stack

Data structures for financial computation

📊 Core Logic

The system applies the following computational process:

Aggregate total expenses.

Calculate equal share per participant.

Compute individual contribution vs expected contribution.

Derive net balances.

Generate optimized debt settlements.

This ensures:

Transparent calculations

Minimal transaction complexity

Accurate balance resolution

🏗 Architecture Concept
User Input → Expense Registration → Balance Calculation Engine → Debt Resolution → UI Display

The logic layer is separated from the presentation layer, allowing future scalability (database integration, authentication, APIs).
